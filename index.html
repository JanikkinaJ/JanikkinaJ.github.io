Turing-machines

Description The turing machine model is not a real machine but rather a model which describes how a Computer would solve a formalized problem. It describes which requirements the algorithm has to ensure the turing machine can correctly execute. Any computer algorithm can be simulated with a turing machine. Parts Tape Infinitely long tape (acts like a memory in a normal computer) Squares on the tape are usually blank at the start The squares can have symbols written on them, in this case blank, zero and one (which is why it is a 3-symbol turing machine) TODO Describe Multi tape turing machines here Head Has a head which is positioned over one of the squares on the tape The head can perform 3 functions: Read the symbol in the current square under the head Edit the symbol (by writing a new one or erasing it) Move the tape left or right by one square so that the machine can read and edit the symbol on a neighbouring square Machine state The machine can have a set of states, the state changes what the machine does with the presented information.
